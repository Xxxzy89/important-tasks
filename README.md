# 重要事务记录 & 闹钟提醒

一个美观、高效的本地任务管理和闹钟提醒网站，采用蓝色、白色和紫色主题设计。

## 功能特性

### 📝 任务管理
- 添加、编辑、删除任务
- 设置任务优先级（高/中/低）
- 设置截止日期和分类
- 任务完成状态标记
- 实时搜索功能

### ⏰ 闹钟提醒
- 自定义闹钟时间和日期
- 重复设置（不重复/每天/每周/每月）
- 多种提醒声音选择
- 闹钟开关控制

### 📊 数据统计
- 任务总数和完成率统计
- 活跃闹钟数量
- 今日重点任务提醒
- 最近活动记录

### 🎨 界面设计
- 响应式设计，支持移动端
- 蓝色、白色、紫色主题
- 现代化UI/UX设计
- 直观的操作界面

## 部署方式

### GitHub Pages（推荐）

1. 将项目上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为发布源
4. 访问 `https://你的用户名.github.io/仓库名`

### 本地运行

1. 直接双击 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (需要安装http-server)
   npx http-server
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: Tailwind CSS CDN
- **存储**: 浏览器 localStorage
- **图标**: Unicode Emoji

## 数据持久化

所有数据保存在浏览器的localStorage中，包括：
- 任务列表
- 闹钟设置
- 活动记录
- 用户偏好

## 使用说明

1. **添加任务**: 点击"添加任务"按钮，填写任务信息
2. **设置闹钟**: 点击"添加闹钟"按钮，设置提醒时间
3. **完成任务**: 点击任务旁的"完成"按钮
4. **搜索任务**: 在搜索框中输入关键词
5. **查看统计**: 查看顶部的统计信息面板

## 注意事项

- 数据仅保存在当前浏览器中
- 清除浏览器数据会丢失所有记录
- 建议定期备份重要数据
- 闹钟提醒需要浏览器标签页保持打开状态
